============
Installation
============

Dependencies
------------
  - Python 3.6+
  - OpenBLAS
  - OpenMP (for ``faiss`` PyPi dist)
  - Zookeeper ensemble


Stable release
--------------

Install Needlestack using pip::

    $ pip install needlestack

To install dependencies for ``faiss`` indices, do::

    $ pip install "needlestack[faiss]"
